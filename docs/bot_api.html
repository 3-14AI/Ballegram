Документация Ballegram Bot API

Добро пожаловать в Ballegram Bot API! Данный API позволяет создавать ботов с возможностями, приближенными к современному Telegram API. Взаимодействие осуществляется по HTTP POST или GET запросам.

Основные типы данных

Все запросы и ответы используют JSON.

Message

{
  "message_id": 123,
  "chat_id": "user-uuid",
  "text": "Hello world",
  "date": 1672531200
}

ReplyKeyboardMarkup / InlineKeyboardMarkup

Позволяют прикреплять клавиатуры к сообщениям для удобного взаимодействия с пользователем.

Регистрация бота

POST /bot/register

Регистрирует нового бота и возвращает токен доступа.

Параметры:

ПолеТипОписание
nameStringИмя нового бота

Установка Webhook

POST /bot/{botId}/webhook

Устанавливает URL для получения входящих обновлений (сообщений).

Отправка сообщений

POST /bot/{token}/sendMessage

Используйте этот метод для отправки текстовых сообщений.

Тело запроса:

ПолеТипОбязательноОписание
chat_idStringДаУникальный идентификатор чата
textStringДаТекст сообщения (до 4000 символов)
reply_markupObjectНетДополнительные опции интерфейса (InlineKeyboardMarkup, ReplyKeyboardMarkup)

Пример:

{
  "chat_id": "12345",
  "text": "Привет!",
  "reply_markup": {
    "inline_keyboard": [[{"text": "Нажми меня", "callback_data": "btn1"}]]
  }
}

Получение обновлений (Long Polling)

GET /bot/{token}/getUpdates

Используйте этот метод для получения входящих обновлений через long polling (как альтернатива webhook).

Query параметры:

  • offset (int): Идентификатор первого возвращаемого обновления
  • limit (int): Ограничение количества (по умолчанию 100)
  • timeout (int): Таймаут в секундах

Установка команд бота

POST /bot/{token}/setMyCommands

Позволяет изменить список команд, поддерживаемых ботом (меню команд).

Тело запроса:

{
  "commands": [
    {"command": "start", "description": "Запуск бота"},
    {"command": "help", "description": "Помощь"}
  ]
}
